# 🎓 UniBrowser: The Future of Campus Exploration

**Live Demo:** [unibrowser.netlify.app](https://unibrowser.netlify.app/)

## 🚀 Project Overview
UniBrowser is a modern, responsive University Portal designed to bridge the gap between prospective students and their future campus. Built with a focus on user experience, it combines high-performance web tech with interactive tools to make university admissions and exploration seamless.

### 🌟 Key Features
- **Smart Course Navigator:** An interactive tool where students can input their Mathematics and Physics marks to receive personalized engineering stream recommendations based on their academic strengths.
- **AI-Powered Chatbot:** A persistent floating assistant to answer immediate queries regarding admissions and campus life.
- **Virtual Campus Tour:** Integrated high-quality video walkthroughs allowing students to explore the campus remotely.
- **Modern UI/UX:** A "Crimson & Cream" aesthetic featuring glassmorphism, smooth Framer Motion animations, and a mobile-first design.

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite
- **Styling:** Tailwind CSS (Modern Utility-first CSS)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Netlify / Vercel
- **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

1.  **Clone the repo:**
    ```bash
    git clone [https://github.com/manankmishrai/uni.git](https://github.com/manankmishrai/uni.git)
    ```
2.  **Navigate to the frontend:**
    ```bash
    cd frontend
    ```
3.  **Install dependencies:**
    ```bash
    npm install --legacy-peer-deps
    ```
4.  **Run locally:**
    ```bash
    npm run dev
    ```
